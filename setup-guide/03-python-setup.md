## Установка на Windows

### Шаг 1: Скачивание Python

1. Перейдите на [python.org/downloads](https://www.python.org/downloads/)
2. Скачайте **Python 3.12.x** (последняя стабильная версия)
3. Сохраните установщик

**[Место для скриншота скачивания Python]**

### Шаг 2: Установка

**ВАЖНО:** Выполните эти шаги точно!

1. Запустите установщик **от имени администратора**
2. ✅ **ОБЯЗАТЕЛЬНО** поставьте галочку:
   ☑️ Add Python.exe to PATH

3. Нажмите **"Install Now"**

**[Место для скриншота установщика с галочкой PATH]**

4. Дождитесь завершения установки
5. Нажмите **"Close"**

### Шаг 3: Проверка установки
