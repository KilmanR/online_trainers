## 1. Подготовка загрузочной флешки

### Шаг 1.1: Запустите Rufus

1. Вставьте USB-флешку (минимум 8 ГБ)
2. Запустите Rufus от имени администратора
3. Выберите вашу флешку в поле "Устройство"

### Шаг 1.2: Настройте Rufus

1. Нажмите **"Выбрать"** и укажите скачанный образ Ubuntu
2. Схема раздела: **GPT**
3. Целевая система: **UEFI (non CSM)**
4. Нажмите **"Старт"**

**[Место для скриншота Rufus]**

### Шаг 1.3: Дождитесь записи

Процесс займет 5-10 минут. После завершения закройте Rufus.

## 2. Подготовка диска

### Шаг 2.1: Откройте Управление дисками

1. Нажмите `Win + X`
2. Выберите **"Управление дисками"**

**[Место для скриншота управления дисками]**

### Шаг 2.2: Освободите место

1. Выберите диск с данными (обычно D: или другой)
2. Нажмите правой кнопкой → **"Сжать том"**
3. Укажите размер: **195000 МБ** (~192 ГБ)
4. Нажмите **"Сжать"**

**Важно:** Не сжимайте системный диск C:!

### Шаг 2.3: Проверьте результат

Должно появиться **"Нераспределенное пространство"** черного цвета.

## 3. Загрузка с флешки

### Шаг 3.1: Перезагрузите компьютер

1. Перезагрузите компьютер
2. Сразу нажимайте клавишу входа в Boot Menu:
   - **F12** (Lenovo, Dell)

   - **F11** (MSI)

## 4. Установка Ubuntu

### Шаг 4.1: Выберите язык

1. Выберите **Русский** (или English)
2. Нажмите **"Установить Ubuntu"**

**[Место для скриншота выбора языка]**

### Шаг 4.2: Раскладка клавиатуры

1. Выберите **Russian** (или English)
2. Нажмите **"Продолжить"**

### Шаг 4.3: Обновления и программы

1. ✅ **Обычная установка**
2. ✅ **Установить стороннее ПО** (галочка внизу)
3. Нажмите **"Продолжить"**

### Шаг 4.4: Тип установки (ВАЖНО!)

Выберите: **"Другой вариант"** (ручная разметка)

**[Место для скриншота выбора типа установки]**

Нажмите **"Продолжить"**

### Шаг 4.5: Разметка диска

Вы увидите список разделов. Найдите **"свободное место"** (которое вы освободили).

#### Создайте разделы:

**Раздел 1: Корневой (/)**
1. Выберите свободное место
2. Нажмите **"+"**
3. Размер: **осталось по умолчанию** (всё свободное)
4. Тип: **Первичный**
5. Местоположение: **Начало**
6. Использовать как: **Журналируемая файловая система Ext4**
7. Точка монтирования: **/**
8. Нажмите **"OK"**

**[Место для скриншота создания раздела]**

**Раздел 2: Подкачка (опционально)**
1. Снова выберите оставшееся свободное место
2. Нажмите **"+"**
3. Размер: **8192 MB** (8 ГБ)
4. Использовать как: **Раздел подкачки**
5. Нажмите **"OK"**

### Шаг 4.6: Загрузчик

Убедитесь, что **"Устройство для установки системного загрузчика"** указывает на:
- **/dev/nvme0n1** (для NVMe SSD)
- или **/dev/sda** (для обычного диска)

**НЕ выбирайте разделы Windows!**

### Шаг 4.7: Начать установку

1. Проверьте, что не форматируете разделы Windows
2. Нажмите **"Установить сейчас"**
3. Подтвердите изменения на диске

## 5. Первоначальная настройка

### Шаг 5.1: Часовой пояс

1. Выберите на карте **Moscow** (или ваш город)
2. Нажмите **"Продолжить"**

**[Место для скриншота выбора часового пояса]**

### Шаг 5.2: Создание пользователя

Заполните поля:

- **Ваше имя**: `Alex Dev` (ваше имя)
- **Имя компьютера**: `alexdev-laptop` (любое, латиницей)
- **Имя пользователя**: `alexdev` (латиницей, без пробелов)
- **Пароль**: придумайте надежный пароль
- **Подтверждение пароля**: введите еще раз

**[Место для скриншота создания пользователя]**

Нажмите **"Продолжить"**

### Шаг 5.3: Ожидание установки

Дождитесь завершения установки (15-30 минут).

**[Место для скриншота процесса установки]**

### Шаг 5.4: Перезагрузка

1. Нажмите **"Перезагрузить"**
2. Когда появится сообщение, нажмите **Enter**
3. Вытащите USB-флешку

### Шаг 5.5: Первый вход

1. Выберите пользователя
2. Введите пароль
3. Нажмите **"Войти"**
